This directory contains our [shields.io](https://shields.io/) badge asset encoded
as a [JSON Endpoint Response](https://shields.io/badges/endpoint-badge).

is freely available and distributed under a [CC0-1.0](https://www.svgrepo.com/page/licensing/#CC0)
